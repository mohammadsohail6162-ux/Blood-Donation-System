# 🩸 Blood Donation System

A web-based Blood Donation Management System developed using **Python, Django, HTML, CSS, JavaScript, and SQLite**. This project helps connect blood donors with recipients and simplifies the blood donation process through an easy-to-use online platform.

## 📌 Project Overview

The Blood Donation System is designed to manage donor registrations, blood requests, user authentication, and blood donation records. The system provides a centralized platform where users can register as donors and request blood when needed.

## 🚀 Features

* User Registration and Login
* Donor Registration Management
* Blood Request Management
* User Profile Management
* Admin Dashboard
* Blood Donation Certificate Generation
* Secure Authentication System
* Responsive User Interface
* Database Management using SQLite

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/mohammadsohail6162-ux/Blood-Donation-System.git
```

### 2. Navigate to Project Directory

```bash
cd Blood-Donation-System
```

### 3. Install Dependencies

```bash
pip install django
```

### 4. Run Migrations

```bash
python manage.py migrate
```

### 5. Start Development Server

```bash
python manage.py runserver
```

### 6. Open Browser

```text
http://127.0.0.1:8000/
```
